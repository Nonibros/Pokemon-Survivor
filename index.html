Welcome to the Project!

Thanks for visiting the Pokémon Survivor main page!

This page holds some quick access links for those who want to play the game and learn more about it.

Use the navigation bar at the top to access the various pages on the site.

How to Play

Download the current patch and then patch it to a vanilla Firered ROM. That's it, you're set from that point! Enjoy!

Search for "Pokemon Fire Red (U)(Squirrels)" for the exact version.

You can also follow the guide here for everything you need to know.

Authors and Contributors
